# 语音大模型（ASR / TTS / 语音对话）

> 从语音识别、语音合成到 GPT-4o 那样的实时语音对话，语音是多模态的重要一环。本文梳理语音任务、表示方式、端到端语音大模型与实时对话架构。

## 一、语音的三类核心任务

| 任务 | 全称 | 做什么 | 代表 |
| --- | --- | --- | --- |
| **ASR** | 自动语音识别 | 语音 → 文字 | Whisper、Paraformer |
| **TTS** | 语音合成 | 文字 → 语音 | VITS、CosyVoice、ChatTTS |
| **语音对话** | — | 语音进、语音出，端到端交互 | GPT-4o 语音、Moshi |

附带能力：**语音克隆**（少量样本复刻音色）、**说话人分离**、**情感/风格控制**。

## 二、语音怎么被模型处理？

语音是连续波形，要先变成模型能处理的形式：

- **声学特征**：传统上转成 **Mel 频谱图（Mel-Spectrogram）**——把波形按时间和频率展开成「图」，再喂给神经网络。
- **离散音频 token**：现代做法用 **音频编解码器（Neural Audio Codec，如 EnCodec、SoundStream）** 把音频压缩成**离散 token**，于是语音可以像文本一样被 Transformer 自回归地建模和生成。这是「语音大模型」统一处理的关键。

## 三、代表模型

- **Whisper（OpenAI）**：在海量多语种数据上训练的 ASR 模型，鲁棒性强、开源，是 ASR 事实标准。架构是 Encoder-Decoder Transformer（音频编码 + 文本解码）。
- **VITS / CosyVoice / ChatTTS**：高质量 TTS，自然度接近真人，支持音色/情感控制。
- **CosyVoice、F5-TTS**：支持零样本语音克隆。

## 四、端到端语音对话大模型

传统语音助手是**级联（Cascade）**：ASR → LLM → TTS 三段串联。问题：延迟高、丢失语音中的情感/语气、错误逐级累积。

**端到端语音大模型**（如 GPT-4o 语音、Moshi）直接「语音进、语音出」，在一个模型里统一处理：

```
级联式：  语音 ─ASR→ 文字 ─LLM→ 文字 ─TTS→ 语音   (慢、丢情感、误差累积)
端到端：  语音 ──────── 统一多模态模型 ────────→ 语音 (低延迟、保留语气情感)
```

优势：**低延迟**（可实时打断、对话自然）、**保留副语言信息**（语气、情感、笑声）、可同时理解和生成语音。实现上常把音频 token 和文本 token 一起喂进多模态 LLM（见 [多模态](/multimodal/multimodal) 的原生多模态思路）。

## 五、实时语音的工程挑战

- **低延迟**：要做到对话级响应（数百毫秒），需流式 ASR/生成、边听边想边说。
- **打断（Barge-in）**：用户能随时打断，模型要能停下来听。
- **全双工**：同时听和说，而非「你说完我再说」。
- **流式处理**：音频是流，要边到边处理，不能等说完。

## 六、高频追问

**Q：语音怎么变成模型能处理的输入？** 两种主流方式：转成 Mel 频谱图（把波形按时频展开成「图」）喂神经网络；或用神经音频编解码器（EnCodec 等）把音频压成离散 token，从而能像文本一样被 Transformer 自回归建模。

**Q：级联式语音助手和端到端语音大模型的区别？** 级联是 ASR→LLM→TTS 三段串联，延迟高、丢失语气情感、误差累积；端到端在一个模型里「语音进语音出」，低延迟、保留副语言信息、可实时打断，体验更自然（如 GPT-4o 语音）。

**Q：Whisper 是什么架构？** Encoder-Decoder Transformer：音频编码器处理 Mel 频谱，文本解码器自回归生成文字。在海量多语种弱标注数据上训练，鲁棒性强，是开源 ASR 的事实标准。

**Q：实时语音对话的主要工程挑战？** 低延迟（流式处理、边听边说）、可打断（barge-in）、全双工（同时听说）。需要流式 ASR/生成和专门的对话调度，而非「说完再处理」。

**Q：语音 token 化有什么意义？** 把连续音频离散成 token 后，语音就能和文本统一在同一个自回归 Transformer 里建模与生成，这是端到端语音大模型（理解+生成语音）得以实现的基础。

**Q：语音克隆怎么做的？** 用音频编解码 + TTS 模型，从少量目标音色样本中提取说话人特征（音色嵌入），在生成时作为条件注入，从而用目标音色合成任意文本。注意其滥用风险（伪造、诈骗），属 [大模型安全](/advanced/safety) 关注范畴。
