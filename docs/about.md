# 关于本项目

## 简介

**LLMGuide（大模型面试指南）** 是一个开源的大模型 / LLM 面试知识库，参考 [JavaGuide](https://javaguide.cn/) 的形式，系统整理大模型方向的「八股文」与高频面试题，覆盖从底层原理到工程落地的完整知识体系：

- 大模型基础：Transformer、Attention、位置编码、Tokenizer
- 训练与微调：预训练、缩放定律、SFT、LoRA/QLoRA、RLHF/DPO
- 应用开发：Prompt 工程、RAG、Agent、Function Calling、MCP
- 工程落地：推理优化与部署、模型评估
- 进阶：多模态、经典模型盘点

## 如何使用

建议按照[首页学习路线](/)循序渐进，也可以直接通过顶部导航、左侧目录或右上角搜索定位到你关心的主题。每篇文章末尾都附有「高频追问」，方便查漏补缺。

## 技术栈

本站基于 [VitePress](https://vitepress.dev/) 构建，纯静态、可一键部署到 GitHub Pages。
